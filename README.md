## 📖 About
ChatDot is a minimalist, real-time chat web app inspired by modern messaging tools. Perfect for personal projects, team collaboration, or learning Socket.io/WebSockets. Built with performance in mind for small-to-medium user bases.

## ✨ Features
- Real-time messaging with instant delivery
- User authentication & private rooms
- Emoji support & rich message formatting
- Responsive design (mobile/desktop)
- Message history & search
- Online/offline status indicators
- Secure Socket.io connections
